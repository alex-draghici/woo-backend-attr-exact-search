=== WC Backend Attribute Exact Search ===
Requires at least: 3.0.1
Tested up to: 3.4
Stable tag: 4.3

Prioritize exact matches when searching for product attribute terms in the backend for terms shorter than 3 characters.

== Description ==

Prioritize exact matches when searching for product attribute terms in the backend for terms shorter than 3 characters.

== Installation ==

1. Upload `woo-backend-attr-exact-search` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
